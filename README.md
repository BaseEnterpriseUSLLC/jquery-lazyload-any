# jQuery-lazyload-any
This is a jQuery plugin provides a lazyload function for everything.

## Demo
[Images](http://emn178.github.io/jquery-lazyload-any/demo/images/)

## Browser Support
jQuery-lazyload-any currently supports IE7+, Chrome, Firefox, Safari and Opera.

## Usage
HTML
```HTML
<div id="#you-want-lazyload">
  <!--
    Anything you want to lazyload
  -->
</div>
```
JavaScript
```JavaScript
$('#you-want-lazyload').lazyload();
```
## Example
HTML
```HTML
<div class=".lazyload">
  <!--
    <img src="image.png" />
  -->
</div>
```
JavaScript
```JavaScript
$('.lazyload').lazyload();
```
